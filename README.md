# Sentiment Analysis on Movie Reviews

This project develops a sentiment analysis system using NLP and machine learning.

## Dataset
IMDb Movie Review Dataset (50,000 reviews)

## Models
- Logistic Regression
- Naive Bayes

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
